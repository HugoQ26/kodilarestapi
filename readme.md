# Kodilla excercise with express.js No3

Simple REST api with 3 endpoints:
/concerts
/seats
/testimonials

Each enpoint can handle this methods:

GET "/"
POST "/"
GET "/:id"
GET "/random"
PUT "/:id"
DELETE "/:id"

All reponses are in JSON also not found.

## Installation

First clone repo.
Next:
`npm install`
And then run:
`node server.js`
